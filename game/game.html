3D Game UI

Dragon Adventures is an adventure game where the user plays as a dragon. The dragon can collect coins to trade for food, weapons, and potions. Additionally, it levels up while exploring all there is to offer in this fantasy world. The arrows exist alongside the dragon since this is a game meant for the tablet and that will be the primary way to navigate. If the user wishes to go diagonally, then the user would be able to click on two directions (i.e. up and left) at the same time to do so.
